Array

l'array e` una variabile multidimensionale. Dentro a una variabile posso inserire un solo valore, dentro un array posso inserire piu` valori.

dentro a un array incontriamo strutture di dati omogenee

dentro a un array possiamo conservare qualunque tipo di dato (primitivi o oggetti)

gli array sono degli elenchi / delle liste dentro a cui salvare diversi tipi di dati

Dentro a un array posso associare piu` valori ad un unico nome di variabile

l'uso degli array ci evita di definire piu` variabili e semplifica lo svolgimento delle operazioni

Proprieta` degli array

in javascript esistono una serie di metodi che si indicano con nome_array.nome_metodo
che ci permettono di compiere una serie di operazioni sul mio array.

La prima proprieta` che vediamo e` length

la proprieta` length conta quanti elementi sono contenuti in un array

Come inserire elementi in un array

per inserire elementi in un array uso un nuovo metodo che si chiama Push

Altri metodi base degli array

pop, shift, unshift
