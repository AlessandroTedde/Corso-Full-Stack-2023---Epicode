Logica Condizionale

Qualunque script in un' ultima analisi e` una sequenza di istruzioni che vengono eseguite dall'engine js
Le nostre istruzioni (tutto quello che scriviamo all'interno di uno script) sono un flusso di comandi
questi comandi stabiliscono il comportamento del sistema (nel mio caso la pagina web) per portarlo da uno stato iniziale a uno stato finale. In altre parole il programma ci permette di descrivere un algoritmo tramite il cui possiamo raggiungere un certo obiettivo oppure un certo risultato.
Per ottenere un risultato avremo bisogno di alcune istruzioni che ci aiutano a stabilire il flusso di esecuzione del nostro programma

blocchi di codice

Piu` istruzioni possono essere raggruppate in blocchi che fungono come una singola istruzione. I blocchi di codice sono importanti quando evidenziamo che un gruppo di istruzioni deve essere eseguito interamente
tipicamente nel codice condizionale eseguiamo dei blocchi di codice in seguito alla valutazione di una condizione
un blocco di codice e` racchiuso tra parentesi graffe

la logica condizionale e` una categoria particolare di istruzioni che consentono di eseguire blocchi di codice alternativi in base a una condizione. Lo script in un certo senso "decide" se eseguire un blocco di codice oppure un altro, la decisione sara` basata sulla verifica della condizione.
JS prevede due istruzioni condizionali

  • if
  • switch

if prevede tre forme

  • if semplice
  • if con alternativa (if...else)
  • if a cascata (if...else if...else)

if semplice

La forma if semplice esegue un blocco di codice solo se una condizione e` vera
se la condizione non e` vera il blocco di codice non viene eseguito

If...else

In questa forma viene eseguito un blocco di codice oppure un altro in base al valore della condizione
