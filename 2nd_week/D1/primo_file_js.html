oggetto console

la console e uno strumento fondamentale per scrivere codice js (per qualsiasi linguaggio di programmazione in realta)

la console ci permette di fare il debug del codice, cioe ci permette di controllare il flusso del nostro codice

spesso la console si usa per tenere traccia del mio script, dell'evoluzione del programma, o verificare dei valori

per vedere la console tastro destro, ispeziona, console

var, let, const

var e' la prima keyword usata da js per usare una variabile, var e' accessibile in qualunque parte dello script

let e' introdotta da EcmaScript6, e' accessibile soltanto in un determinato ambito (scope) che e' delimitato da un blocco di codice

Un blocco di codice js e' una serie di istruzioni delimitate da {.. blocco di codice}

const

Introdotte da EcmaScript6 -> istruiscono l'engine che il valore assegnato non e' modificabile
