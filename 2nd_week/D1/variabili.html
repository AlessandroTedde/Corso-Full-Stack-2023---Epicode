le variabili

servono per memorizzare dei valori

ogni variabile sara identificata da un nome che segue alcune regole:

  • niente spazi, niente trattini - e niente caratteri speciali(?, . ecc.)
  • deve per forza iniziare con una lettera oppure con $ o _
  • sono case-sensitive
  • non puo iniziare con un numero
  • non deve mai coincidere con le keyword di Javascript (if, else, switch, return, false)

dichiarazione implicita delle variabili

non e obbligatorio dichiarare le variabili, cioe fare un'operazione esplicita di creazione
quando c'e l'identificatore = l'engine crea implicitamente la variabile se non esiste gia

strict mode

e un'opzione di controllo del codice, EcmaScript 5 -> se viene eseguito lo strict mode javascript viene eseguito osservando delle regole di scrittura piu strette -> es. ci arriva un codice di errore quando non dichiariamo le variabili. Nell'esecuzione normale javascript ci da' delle regole piu' permissive che se dichiariamo lo strict mode ci obbligano a uno stile di prorammazione piu' strutturato
