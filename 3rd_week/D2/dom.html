DOM

Document Object Model e` in realta` una proprieta` dell'oggetto window e rappresenta l'intera struttura html caricata nella mia pagina ha una struttura a oggetto. Il DOM fornisce una rappresentazione gerarchica degli oggetti, che viene spesso chiamata Dom tree (assomiglia a un albero genealogico). Questo albero di oggetti e` accessibile tramite JavaScript, infatti possiamo fare una serie di cose

  • leggere gli elementi del DOM
  • modificare la struttura degli elementi (tag)
  • modificare il contenuto degli elementi (tag)
  • modificare lo stile

Il DOM si presenta come un albero con una serie di nodi che hanno diversi tipi

Manipolare il DOM via JavaScript

  1. I metodi per selezionare un elemento del DOM
  2. modificare lo "style" di un elemento
  3. modificare le classi di un elemento
  4. Modificare il contenuto di un elemento
  5. Creare, inserire e eventualmente rimuovere degli elementi nel DOM
